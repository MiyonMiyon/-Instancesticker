# About #MiyonTicker
- [#InstanceTicker](https://github.com/MiyonMiyon/InstanceTicker) で培ったものを PHP で公開予定。
- フォークしたら「#なんとかTicker」を配布し合う構図？
- アイコン画像データは収録していません。
- - SNSソフトウェアのアイコンはこちらで作成した代替アイコンが設定されています。
- - 「34.wtf 画像にする」を有効にすると、#InstanceTicker と同一になります。
